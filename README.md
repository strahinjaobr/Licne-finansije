# Lične Finansije

Java Swing desktop aplikacija za praćenje ličnih finansija sa višekorisničkom podrškom. Aplikacija je jednostavna, bez baze, sa serijalizacijom. U budućnosti će biti dodato još funkcionalnosti.

## Funkcionalnosti

- Registracija i prijava korisnika sa validacijom podataka
- Dodavanje, izmena i brisanje prihoda i troškova
- Pretraga transakcija po opisu u realnom vremenu
- Filtriranje transakcija po datumu i kategoriji
- Prikaz stanja računa sa opcijom sakrivanja iznosa
- Analiza prihoda i troškova sa pie dijagramom po kategorijama
- Keyboard shortcuts za brži rad

## Keyboard Shortcuts

- `+` — Dodaj prihod
- `-` — Dodaj trošak
- `Ctrl + I` — Izmeni transakciju
- `Ctrl + D` — Obriši transakciju
- `Enter` — Prijava

## Tehnologije

- Java 25
- Java Swing (GUI)
- Maven
- JFreeChart (dijagrami)
- Serijalizacija (čuvanje podataka)

## Pokretanje

1. Kloniraj repozitorijum
